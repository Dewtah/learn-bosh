BOSH CLI is a command line interface to the Director. Follow these instructions to install it.

Point your CLI to the bosh-lite environment, saving it with the alias lite. But first save the CA certificate to access bosh-lite Director:

$ curl -O https://raw.githubusercontent.com/cloudfoundry/bosh-lite/master/ca/certs/ca.crt

$ bosh -e 192.168.50.4 alias-env lite --ca-cert ca.crt

Using environment '192.168.50.4' as anonymous user Name Bosh Lite Director UUID ... Version 260.0.0 (00000000) CPI warden_cpi Features compiled_package_cache: enabled dns: disabled snapshots: disabled User (not logged in) Succeeded

Log in using admin/admin:

$ bosh -e lite login

Username (): admin Password (): Using environment '192.168.50.4' as client 'admin' Logged in to '192.168.50.4' Succeeded

Now we are ready to deploy!
