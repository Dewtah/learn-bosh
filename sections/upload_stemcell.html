A Stemcell is an operating system image that BOSH uses to create VMs. Official BOSH stemcell downloads are maintained with security updates at bosh.io.

Upload stemcell to Director.

$ bosh upload stemcell https://s3.amazonaws.com/bosh-warden-stemcells/bosh-stemcell-2776-warden-boshlite-ubuntu-trusty-go_agent.tgz
